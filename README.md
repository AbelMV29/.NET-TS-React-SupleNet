# SupleNet

Este repositorio contiene dos proyectos principales: el Backend y el Frontend de la aplicación SupleNet.

## Tecnologías Usadas

### Backend
- .NET 8.0
- Entity Framework Core
- MediatR + CQRS
- FluentValidation
- CloudinaryDotNet
- Swashbuckle (Swagger)

### Frontend
- React
- TypeScript
- Vite
- ESLint
- TailwindCSS
- Axios
- Zod
